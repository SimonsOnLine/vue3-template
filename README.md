  [EditorConfig](https://editorconfig.org/) + [Prettier](https://prettier.io/) + [ESLint](https://eslint.org/) + [Airbnb JavaScript Style Guide](https://github.com/airbnb/javascript)

### 开发使用

##### 运行项目

- `npm i`
- `npm i commitizen -g`

#### 代码提交

- git add xx
